# FORMAT PRACTICE

Textio has asked you to log individual users and their recent messages.

## ASSIGNMENT

Finish the `userLog` variable. It should create the string:

```
Name: FNAME LNAME, Age: AGE, Rate: MESSAGERATE, Subscriber: ISSUBSCRIBED, Message: MESSAGE
```

Where `FNAME` `LNAME` `AGE` `MESSAGERATE` `ISSUBSCRIBED` and `MESSAGE` correspond to the variables above. `messageRate` should be rounded to the `tenths` place!
